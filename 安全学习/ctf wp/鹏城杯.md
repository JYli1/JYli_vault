# 【pcb5-ez_php】
一个登录页面，账号密码不知道，抓包看一下，发现cookie信息base64解密后是php序列化字符串，构造对应脚本
```php
<?php  
  
namespace Session;  
  
class User {  
    private $username = "admin";  
}  
  
$obj = new User();  
  
echo base64_encode(serialize($obj));
```
